# POKEMON TCG

## Quick Setup

1. Clone the repository:
```bash
git clone https://github.com/thureintun-me/pokemon-tcg
cd pokemon-tcg
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the server:
```bash
npm start
# or
yarn start
```

4. Run the app:
```bash
# For Android
npm run android
# or
yarn android

# For iOS
npm run ios
# or
yarn ios
```
